How does MonkeyType Enhancer work?

How to set up MonkeyType Enhancer: The only thing you need to do is go to settings and turn quick reset to enter. Make sure you use this after finishing a test and going to the next since MonkeyType Enhancer tracks the enter key to know when you finished a test. Also, make sure the session timer starts at zero. This can be done by reloading the page.

Everything is automated by MonkeyType Enhancer to make it straighforward for the user. There are 5 phases spread out over 15 minutes of typing time:

  1. Testing (0-5): Your accuracy from this phase determines the letter that you practice in the next phase. You can check your stats in the stats tab at the end of this phase. The language used is english 5k.
  2. Training (5-10): The letter that you missed the most during testing will be in every word in this phase for optimal practice. The language used is english 5k.
  3. Practice 1 (10-13): This phase practices a smaller set of more common words to help develop muscle memory. The language used is english 1k.
  4. Practice 2 (13-15): This phase uses a set of the 200 most commonly used words. You will eventually develop muscle memory for these words and type them faster, allowing you to practice typing at a faster speed.
  5. Ending (15+): It is recommended to finish on a test with 100% accuracy to end on a high note.

FAQs

What are the different languages in MonkeyType? MonkeyType has different word groups that are larger than the default for the language. The default includes 200 words which is not enough to practice for real world uses, so MonkeyType Enhancer uses larger word groups. You can change these manually by pressing escape and typing "language".

What is the best time length for the tests? This varies person to person, but for most people 15 seconds most directly translates to the real world since most people type in short bursts, not long periods of time.
